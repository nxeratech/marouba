## THE SOLUTION

**Label:** THE SOLUTION

**Headline:**
Teach once. Replay forever. Correct once.

**Body:**
Marouba works with any app on Windows or Mac. Photoshop. Ableton. AutoCAD. Excel. Blender. Your terminal. Your browser. Anything you can do, Marouba can remember. It does not guess. It does not look at your screen and hope. It knows — because you showed it once.

**Three cards (keep existing Teach / Replay / Repair layout):**

Teach
Perform your workflow normally. Marouba observes the context, the route, and the intent behind what you did. Stored as human-readable Markdown in a vault that belongs to you.

Replay
Need the same workflow again? Marouba tries every known route in order — cheapest first. API, scripting, UI elements, keyboard shortcuts. If one breaks, it falls back to the next.

Repair
When something moves or breaks, Marouba asks you to correct it once, records the fix, and never asks again. The vault updates. The next run just works.
