The problem

The frontier labs are building smarter models. Nobody is building memory with real human taste.

AI tools that look at your screen are slow, expensive, and learn nothing. Every session starts from zero. Every workflow re-explained. Every fix forgotten. The industry's answer is bigger models and longer context windows. But the problem is not intelligence. It is memory.

Every workflow is re-taught

The average knowledge worker repeats the same digital workflows dozens of times a month. AI agents start from scratch every single time.

Every update breaks automation

A button moves, a dialog changes, a version updates. Every fragile pixel-click loop breaks completely and silently.

Every fix disappears

When something breaks and you fix it manually, that fix vanishes. It never becomes shared knowledge. You do it again next time.

The solution

Teach once. Replay forever. Correct once.

Marouba works with any app on Windows or Mac. Photoshop. Ableton. AutoCAD. Excel. Blender. Your terminal. Your browser. Anything you can do, Marouba can remember. It does not guess. It does not look at your screen and hope. It knows - because you showed it once.

Teach

Perform your workflow normally. Marouba observes the context, the route, and the intent behind what you did. Stored as human-readable Markdown in a vault that belongs to you.

Replay

Need the same workflow again? Marouba tries every known route in order - cheapest first. API, scripting, UI elements, keyboard shortcuts. If one breaks, it falls back to the next.

Repair

When something moves or breaks, Marouba asks you to correct it once, records the fix, and never asks again. The vault updates. The next run just works.

The vault

Not a macro. Not a script. A living record of how you work.

Traditional automations store actions: click here, type this, press Enter. Marouba vaults store every route - element names, shortcuts, commands - as human-readable Markdown, plus fallback paths, verification rules, and repair history.

Human readable. Editable directly. Portable. Yours.

Git-friendly. Obsidian-compatible.

Shareable with any agent in your stack - Barry, Claude, n8n, anything with an HTTP call.

A value system built into the architecture.

---
id: photoshop-export-for-web
app: Photoshop
routes:
  - type: cli
    command: "photoshop.exe -r export.jsx"
  - type: uia
    element: Save for Web
  - type: shortcut
    keys: [ctrl, alt, shift, s]
fallback_order: [cli, uia, shortcut, ask]
verification:
  type: file_exists
  path: "{output_path}"
---
