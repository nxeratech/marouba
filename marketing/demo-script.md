## Production Notes

- **Visual style:** Clean, minimal. Dark mode desktop. Marouba overlays in a distinct accent colour.
- **Music:** Subtle, ambient — don't overpower the narration. Think lo-fi beats or quiet synth pad.
- **Pacing:** The repair sequence (0:20-0:32) is the hero moment. Give it room. Slow down slightly so viewers see the detection → search → fix clearly.
- **Screen recording:** Use actual desktop, not simulated. Real workflows land harder than animations.
- **The vault file cameo:** Show the actual YAML briefly at 0:10 and 0:28. Engineers will pause and read it. Make it real.
- **CTA:** Keep the final card up for 3-4 seconds. Don't rush it.
